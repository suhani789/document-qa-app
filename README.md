# 📄 Document Q&A App

> *"Tired of reading long PDFs manually? Just ask questions!"*

Upload any document and chat with it using Google Gemini AI — powered by RAG (Retrieval Augmented Generation).

## ✨ Features

| Feature | Description |
|---|---|
| 📁 Document Upload | Supports PDF, TXT, DOCX |
| 🤖 AI Q&A | Ask anything about your document |
| 🔍 RAG Pipeline | Finds relevant content before answering |
| 🌐 Web UI | Clean Gradio interface |

## 🛠️ Tech Stack

- Python 3.12
- LlamaIndex (RAG Framework)
- Google Gemini AI
- HuggingFace Embeddings
- Gradio (Web UI)

## 🚀 Run It Yourself

```bash
git clone https://github.com/suhani789/document-qa-app.git
cd document-qa-app
pip install -r requirements.txt
python app.py
```

## 🔮 Future Scope
